# Dashboard

A modern dashboard application built with React, Vite, Tailwind CSS, and DaisyUI.

1. **Clone Repository**
   ```terminal
   git clone https://github.com/MdRehanAli/dashboard.git
   ```


2. **Install dependencies**
   ```terminal
   npm install
   ```

3. **Start the development server**
   ```terminal
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` (or the port shown in your terminal)
